Image Zoom

Zoom into image for a lot more of details.

To use zoom on images, apply a with-zoom class to a <div> element. Inside the <div> element, create another <div> element with the class of magnify, and in addition, place the desired image by using a <img> element with the class of small. The width and height of the image should not be the full width and height of the image.

Do not abuse this feature!

Please use responsibility. Not all images should have the ability to zoom. Use for large images that contain many small hard-to-see details.

Not optimized for mobile touch devices!

Hover feature is not supported in mobile touch devices.

Code

<div class="with-zoom">
	<div class="magnify"></div>
	<img class="small" src="src/img/imagezoomsample.jpg" width="764" height="430" />
</div>
