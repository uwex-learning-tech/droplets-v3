Block-Level Callouts

You would not want to miss it!

Horizontal

Horizontal callout uses a <div> element with the callout class. For callout heading, use <h4> element. For the content, use <p> element. For different variations of callout, append any of the following classes: success, info, warning, or danger to the <div> element's existing class attribute.

Notes: icons can be used with callouts. Please refer to the icons page for a complete list of available icons.

Please refer to the best practice guide for the correct use of icons.

Code

<div class="callout">
	<h4>Something!</h4>
	<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Repellendus,
	recusandae impedit quod consectetur corporis neque suscipit laboriosam eum.
	Neque sit ipsa est consequatur illum optio reprehenderit atque deserunt.</p>
</div>

</div>

Vertical

Horizontal becomes vertical by applying the vertical class. Vertical callouts are best used for columned layouts or floatings.

Code

<div class="three-column">

	<section>
	
		<div class="callout info vertical">
			<h4><span class="icon-reading"></span> Reading</h4>
			<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.
			Repellendus, recusandae impedit quod consectetur corporis neque
			suscipit laboriosam eum. Neque sit ipsa est consequatur illum optio
			reprehenderit atque deserunt facere quo.</p>
		</div>
		
	</section>

	<section>
	
		<div class="callout warning vertical justify">
			<h4><span class="icon-assignment"></span> Assignment</h4>
			<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.
			Repellendus, recusandae impedit quod consectetur corporis neque
			suscipit laboriosam eum. Neque sit ipsa est consequatur illum optio
			reprehenderit atque deserunt facere quo.</p>
		</div>
		
	</section>

	<section>
	
		<div class="callout danger vertical">
			<h4><span class="icon-discussion"></span> Discussion</h4>
			<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.
			Repellendus, recusandae impedit quod consectetur corporis neque
			suscipit laboriosam eum. Neque sit ipsa est consequatur illum optio
			reprehenderit atque deserunt facere quo.</p>
		</div>
		
	</section>
	
</div>

Floating callout is for text wrap purposes. Only work with vertical callouts, and the width is fixed.
