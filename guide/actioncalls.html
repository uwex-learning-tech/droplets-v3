Action Calls

Action Calls only work within D2L courses as long the URL does not change.

Table of Contents

To go to the table of content, add onclick="gotoToC();" to <a> or <button> element.

<p><a class="btn default block" href="javascript:void(0)" onclick="gotoToC();">Table of Contents</a></p>

See the Pen D2L Table of Contents Action Call by Ethan Lin (@Lin87) on CodePen.

Grades

To go to the grades page, add onclick="gotoGrades();" to <a> or <button> element.

<a class="btn success block" href="javascript:void(0)" onclick="gotoGrades();">Grades</a>

See the Pen D2L Grade Action Call by Ethan Lin (@Lin87) on CodePen.

Dropbox

To go to the dropbox page, add onclick="gotoDropbox();" to <a> or <button> element.

Notes: only works when in student mode.

<a class="btn info block" href="javascript:void(0)" onclick="gotoDropbox();">Dropbox</a>

See the Pen D2L Dropbox Action Call by Ethan Lin (@Lin87) on CodePen.

Discussions

To go to the discussion page, add onclick="gotoDiscussions();" to <a> or <button> element.

<a class="btn warning block" href="javascript:void(0)" onclick="gotoDiscussions();">Discussions</a>

See the Pen D2L Discussion Action Call by Ethan Lin (@Lin87) on CodePen.
