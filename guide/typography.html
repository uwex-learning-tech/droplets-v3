Typography

Look matters. Every typeface is unique.

Headings

Headings are used to show the document structure!

See the Pen Headings by Ethan Lin (@Lin87) on CodePen.

Lead Body Copy

A lead body copy is a sentence or paragraph that stands out among the rest of the contents. To create a lead, apply the lead class to a <p> element.

See the Pen Lead by Ethan Lin (@Lin87) on CodePen.

Body Copy

The content of the page using the <p> element.

See the Pen Body Copy by Ethan Lin (@Lin87) on CodePen.

Drop Cap

A large initial letter (or icon) that drops below the first line of a paragraph, usually used at the beginning of a section. Apply the class dropcap to the <p> element to achieve this style.

See the Pen Drop Cap by Ethan Lin (@Lin87) on CodePen.

Hyperlink (or Link)

Create a hyperlink between documents or websites by using the <a> element.

See the Pen Hyperlink by Ethan Lin (@Lin87) on CodePen.

Blockquote

For quoting blocks of content from another source. Can also be use for quoting a passage from a famous person or literature.

Basic

See the Pen Basic Blockquote by Ethan Lin (@Lin87) on CodePen.

With Author or Source

Add an author or a source to the blockqoute by nesting a <small> element after the content.

See the Pen Blockquote with Author or Source by Ethan Lin (@Lin87) on CodePen.

Right Aligned

Right aligned the blockquote by applying the pull-right class.

See the Pen Right Aligned Blockquote by Ethan Lin (@Lin87) on CodePen.

Alternative

An alternative style by applying the quote class.

See the Pen Alternative Blockquote by Ethan Lin (@Lin87) on CodePen.

Lists

Unordered

Create list of items in no specific order by using the <ul> element.

See the Pen Unordered List by Ethan Lin (@Lin87) on CodePen.

Nested Unordered

Create nested list of items in no specific order by nesting the <ul> elements.

See the Pen Nested Unordered List by Ethan Lin (@Lin87) on CodePen.

Ordered

Create list of items in a precise order or steps by using the <ol> element.

See the Pen Ordered List by Ethan Lin (@Lin87) on CodePen.

No Bullets

Create a list of items without bullet style by applying the no-bullet class. All nested <ol> and <ul> elements will not have bullets as well.

See the Pen No Bullets List by Ethan Lin (@Lin87) on CodePen.
