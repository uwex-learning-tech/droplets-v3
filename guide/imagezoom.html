Image Zoom

Zoom into an image for a lot more detail.

To use zoom on images, apply a with-zoom class to a <div> element. Inside the <div> element, create another <div> element with the class of magnify, and in addition, place the desired image by using an <img> element with the class of small. The width and height of the image should not be the full width and height of the image.

Please use responsibly. Not all images should have the ability to zoom. Use for large images that contain many small hard-to-see details.

Accessibility: Image zoom is not optimized for accessibility. Please avoid using image zoom if accessibility is a need.

The hover feature is not supported on mobile devices.
