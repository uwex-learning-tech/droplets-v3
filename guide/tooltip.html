Tooltip

A short message appears when the mouse cursor is positioned over an icon, image, hyperlink, or text in page content. Do not use this component for popovers.

Tooltip has five options: default, top, bottom, right, and left. Each option will be explained below with code and examples. To use tooltip, add the with-tooltip class and the title attribute to the HTML elements including <span>, <a>, <img>, etc. The title attribute is used to hold the tooltip message. Thus, it is required.

Tip: use tooltip for non-essential information. Tooltip adds nice interaction to the content, but excessive use will cause unintended learning problems.

Notes: the <abbr> element does not require the with-tooltip class. When using the <abbr> element, please follow the standards for using an abbreviation.

Default

By default, the tooltip will be displayed on the top of the element without any arrow indication.

See the Pen Tootip by Ethan Lin (@Lin87) on CodePen.

Top

The tooltip will be placed at the top of the text and with an arrow pointing to the element. To specify the top position, add the top class.

See the Pen Tooltip Top by Ethan Lin (@Lin87) on CodePen.

Bottom

The tooltip will be placed at the bottom of the text and with an arrow pointing to the element. To specify the bottom position, add the bottom class.

See the Pen Tooltip Bottom by Ethan Lin (@Lin87) on CodePen.

Right

The tooltip will be placed at the right of the text and with an arrow pointing to the element. To specify the right position, add the right class.

See the Pen Tooltip Right by Ethan Lin (@Lin87) on CodePen.

Left

The tooltip will be placed at the left of the text and with an arrow pointing to the element. To specify the left position, add the left class.

See the Pen Tooltip Left by Ethan Lin (@Lin87) on CodePen.

Other Examples:

Tooltip can also be used on hyperlink and image.

See the Pen Tooltip Examples by Ethan Lin (@Lin87) on CodePen.

Tooltip for image will not be displayed when printed. This is a limitation of CSS3. Avoid using image with tooltip for essential information. Use image caption instead.
