Layouts

A house is built with a blueprint. Same goes for a web page.

Two-Column

A two-column layout uses two nested <section> elements inside a <div> element with two-column class. A <div> element with clearfix class is needed right after the two-column <div> element.

A two-column layout with wide right content column and a left side bar.

Three-Column

A three-column layout uses three nested <section> elements inside a <div> element with three-column class. A <div> element with clearfix class is needed right after the three-column <div> element.

See the Pen Three-Column by Ethan Lin (@Lin87) on CodePen.

First Line Indentation

Indent the first line of a paragraph by applying the indent class to the <p> element.

Notes: the indent class can also be applied to a <div> element. The first line of all the paragraphs inside the <div> element will be indented.

See the Pen First Line Indentation by Ethan Lin (@Lin87) on CodePen.

All Line Indentation

Indent all lines in a paragraph by applying the indent-all class to a <p> element.

Notes: the indent-all class can also be applied to a <div> element. All paragraph lines nested inside the <div> element will be indented.

See the Pen All Line Indentation by Ethan Lin (@Lin87) on CodePen.

Hanging Indentation

Indent all lines except the first line of a paragraph by applying the hanging-indent class to a <p> element.

Notes: the hanging-indent class can also be applied to a <div> element. Any elements nested inside the <div> element will have the hanging indentation.

See the Pen Hanging Indentation by Ethan Lin (@Lin87) on CodePen.

Text Alignments

Align text on the page by applying the left, center, or right class.

See the Pen Text Alignments by Ethan Lin (@Lin87) on CodePen.

Justify Alignment

Justify columns of paragraph by using the justify class. The justify class can be used with individual <div> and <p> elements.

See the Pen Justify Alignment by Ethan Lin (@Lin87) on CodePen.

Footer

Footer is always the last layout at the bottom of the page. The texts in the footer are optional, and it is added by the CSS. By default, the texts will be displayed. To remove the texts, add the no-text class to the <footer> element.

Do not remove the <p> element. The <p> element must stay even if it is empty.
